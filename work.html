Mary's Projects

myFlix

An app that allows users to view information about different movies including Description, Genre and Director. Users can sign up for an account and add favorite movies to their profile. It utilizes password hashing to encrypt users’ passwords. This project was built using the MERN stack (MongoDB, Express, ReactJS, Nodejs) and styled with Bootstrap.


Meetup

An app was created with React, it is a Progressive Web App allowing it to function as a stand-alone app. The main purpose of this app is to find different events in your area. It uses your location to find events close to you or you can search for a city worldwide. AWS Lambda and OAuth are used to obtain authorization for the Meetup API. Charts were added from ReCharts to display the number of events in the next 7 days and RSVP status for events. Unit testing, integration testing, and user acceptance were preformed using jest to ensure scenarios and features functioned as expected.
